\subsection{Responsiveness}
The portfolio is able to be hosted both in a mobile phone and in a pc.This was achieved using an @media tag in the CSS and placing the max-width to be 900 px.This allows the elements to maintain their position in a smaller screen.And when loaded in a pc it ignores the command.Furthermore I used the flex box in my CSS to allow the elements to share space evenly and have a responsive design.\\
\section{STRENGTHS AND WEAKNESSES}
\subsection{Strengths}
The portfolio is highly responsive: it has working links and can work on both mobile phones and in a pc.The portfolio is also aesthetically pleasing , using a moderate color palate that is soft on the eyes.The portfolio also has a similar theme to my CV, hence there is no format change making it easier for the brain to register information.
